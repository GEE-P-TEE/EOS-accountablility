# AI Coder App Prompt: Accountability Chart Maker

I want to build a web-based EOS accountability chart creation platform that helps business owners implementing EOS create, manage, and share visual organizational charts. It should feel professional, intuitive, and collaborative while being optimized for web browsers.

## Project Name:
Accountability Chart Maker

## Target Audience:
• Business owners implementing EOS in their companies
• Entrepreneurs running ecommerce businesses
• Amazon sellers organizing their teams
• Small to medium business leaders adopting EOS methodology

## Core Features and Pages:

### Homepage
• Clean landing page explaining EOS accountability charts and the tool's benefits
• Call-to-action to start creating charts or sign up for an account
• Brief overview of key features: drag-and-drop, collaboration, templates, PDF export

### Dashboard
• Overview of all user's accountability charts
• Quick access to create new charts from templates or from scratch
• Recent activity and collaboration notifications
• Chart sharing management (active shared links, team member access)

### Chart Builder (Main Feature)
• Interactive drag-and-drop interface for creating organizational charts
• Visual hierarchy with connecting lines showing reporting relationships
• Seat/role cards containing: role name, person's name, responsibilities, measurables/Rocks with target numbers
• Dynamic add/remove functionality for roles and seats
• Real-time collaboration with team members
• Customizable colors and styling options
• Auto-save functionality with Supabase backend

### Templates Library
• Pre-built EOS organizational structures for different business types
• Ecommerce business templates (marketing, operations, fulfillment teams)
• Amazon seller templates (PPC, inventory, customer service roles)
• Generic small business templates
• Ability to save custom charts as personal templates

### Collaboration Hub
• Team member management (invite, remove, set permissions)
• Real-time editing with multiple users
• Comment system for feedback on specific roles/seats
• Activity log showing who made what changes
• Role-based permissions (view-only, editor, admin)

### Export & Sharing
• PDF export with professional EOS-style formatting
• Shareable read-only links with customizable permissions
• Print-optimized layouts
• Image export options (PNG, SVG)

## Tech Stack (Recommended Defaults):
• Frontend: Next.js 14, Tailwind CSS, shadcn/UI
• Backend & Storage: Supabase (with real-time subscriptions for collaboration)
• CMS: Notion for help documentation and templates
• Auth: Email/password login with optional Google OAuth
• Additional: React DnD for drag-and-drop functionality, jsPDF for PDF generation

## Design Preferences:
• Font: Inter
• Colors:
   - Primary: #2563eb (EOS blue)
   - Accent: #059669 (success green)
   - Backgrounds: clean white, soft grays (#f8fafc, #f1f5f9)
• Layout: Mobile-first, card-based interface with clean whitespace
• Chart styling: Professional boxes with subtle shadows, customizable accent colors
• Responsive design optimized for desktop chart creation

## Database Schema Considerations:
• Charts table (id, name, owner_id, created_at, updated_at, template_id)
• Seats table (id, chart_id, role_name, person_name, responsibilities, position_x, position_y)
• Measurables table (id, seat_id, description, target_value, current_value)
• Chart_collaborators table (chart_id, user_id, permission_level)
• Templates table (id, name, category, structure_data)

## Key User Flows:
1. Sign up → Choose template or start from scratch → Build chart with drag-and-drop → Invite team members → Export/share
2. Existing user → Dashboard → Open chart → Collaborate with team → Track measurables progress → Export updates
3. Template usage → Browse templates → Customize for business → Add team members and real data → Share with organization
